# WhoCommunity Project
💻 익명으로 자유롭게 소통할 수 있는 커뮤니티 사이트 입니다.

사람들에게 정보 공유를 하거나 의견을 물어볼 수 있고, 그 글에 대하여 댓글로 여러 가지 의견을 주고받을 수 있습니다.
